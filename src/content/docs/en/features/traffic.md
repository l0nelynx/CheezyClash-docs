## Home (desktop)

When connected:

- **Download / Upload rates** appear in the connection hero (live from Mihomo connection totals).
- **Session** card shows cumulative upload/download for the current session.
- **Subscription** card shows profile quota when `subscription-userinfo` is available.

## Proxies

Latency badges reflect the core’s last health-check result per node.

## Android

Traffic and connection stats follow the Android UI; subscription quota uses the same headers as desktop.
