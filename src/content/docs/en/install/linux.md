Desktop Linux packages are built in CI alongside Windows releases. Check [GitHub Releases](https://github.com/l0nelynx/CheezyClash/releases/latest) for AppImage, deb, or zip artifacts for your architecture (`x64`, `arm64`).

## Quick steps

1. Download the Linux artifact from Releases.
2. Install or extract per artifact type.
3. Run CheezyClash — **Proxy mode** is the usual first test; TUN depends on platform capabilities and helper support.

For development builds, see the [desktop README](https://github.com/l0nelynx/CheezyClash/tree/main/desktop) in the main repository.
