## Download

Get the APK for your CPU architecture from [GitHub Releases](https://github.com/l0nelynx/CheezyClash/releases/latest):

| ABI | Typical devices |
| --- | ---------------- |
| `arm64-v8a` | Most phones (recommended) |
| `armeabi-v7a` | Older 32-bit ARM |
| `x86_64` | Emulators, some tablets |

## Install from APK

1. Download the matching APK.
2. Allow installation from unknown sources if prompted.
3. Open CheezyClash and grant VPN permission when connecting.

## Requirements

- **Minimum:** Android 9 (API 28)
- **Recommended:** recent Android with current security patches

## First run

1. Add a subscription URL or import a profile file.
2. Tap connect — Android shows the VPN consent dialog.
3. Select proxy groups and nodes in the Proxies screen.

## Subscription headers

Remote subscriptions can send Mihomo/Clash headers the app understands, including:

- `profile-title`, `announce`
- `profile-update-interval` (auto-update hours)
- `subscription-userinfo` (`upload`, `download`, `total`, `expire`)

## Updates

Install a newer APK over the existing app, or use your distribution channel when published.
