## What is CheezyClash?

CheezyClash is an open-source graphical client for [Mihomo](https://github.com/MetaCubeX/mihomo) (Clash.Meta). It supports Clash YAML profiles, remote subscriptions, proxy groups, routing rules, and traffic monitoring on **Android** and **Windows desktop** (Electron).
