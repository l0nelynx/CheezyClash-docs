CheezyClash — **оболочка для Mihomo**, а не VPN-сервис. Вы добавляете Clash/Mihomo конфиг (подписку или файл); приложение запускает ядро и даёт привычный UI.

## Платформы

| Платформа | UI | Ядро |
| --------- | -- | ---- |
| Android | Kotlin | Mihomo в VPN-сервисе |
| Windows desktop | Electron + React | `mihomo.exe` + helper для TUN |

## Вкладки desktop

| Вкладка | Назначение |
| ------- | ---------- |
| **Home** | Подключение, квота, быстрый выбор сервера |
| **Proxies** | Группы, задержки, Test |
| **Profiles** | Импорт и обновление подписок |
| **Settings** | Proxy/TUN, Access Control |
| **Logs** | Лог Mihomo |
