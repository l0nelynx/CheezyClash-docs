## Quick start (desktop)

1. Install from [GitHub Releases](https://github.com/l0nelynx/CheezyClash/releases/latest):
   - Windows — [`CheezyClash-win-x64.exe`](https://github.com/l0nelynx/CheezyClash/releases/latest/download/CheezyClash-win-x64.exe)
   - macOS — [`CheezyClash-mac-arm64.dmg`](https://github.com/l0nelynx/CheezyClash/releases/latest/download/CheezyClash-mac-arm64.dmg) (Apple Silicon) or [`CheezyClash-mac-x64.dmg`](https://github.com/l0nelynx/CheezyClash/releases/latest/download/CheezyClash-mac-x64.dmg) (Intel)
   - Linux — AppImage / `.deb` on the [releases page](https://github.com/l0nelynx/CheezyClash/releases/latest)
2. Open **Profiles** → paste subscription URL → **Import** → **Activate**.
3. **Settings** → choose **Proxy** (system proxy) or **TUN** (full tunnel; needs helper on Windows).
4. **Home** → **Connect**. Pick a server on **Proxies** or from the server card on Home.

## Quick start (Android)

1. Install the APK matching your CPU (`arm64-v8a` for most phones).
2. Add subscription or import profile.
3. Tap connect and accept the Android VPN permission.

## What you need

| Item | Notes |
| ---- | ----- |
| Mihomo profile | Clash YAML subscription URL or local file |
| Windows x64 | Desktop; helper service for TUN |
| macOS 11+ | Apple Silicon or Intel `.dmg` / zip from Releases |
| Linux x64 / arm64 | AppImage, deb, or zip |
| Android 9+ | VPN permission required to connect |

CheezyClash is **not** a VPN provider — it runs **your** Mihomo config and connects to **your** proxy servers.
